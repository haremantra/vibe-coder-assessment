The Architect’s Fix: Pre-Execution Triage

Two fields that change everything

def triage_failure_response(task, failure_event):
    rev  = task.reversibility
    crit = task.criticality

    if not rev and crit in ("high", "critical"):
        return Action.HALT_AND_ESCALATE_TO_HUMAN
    if not rev and crit == "medium":
        return Action.REQUIRE_HUMAN_APPROVAL_BEFORE_RETRY
    if rev and crit in ("high", "critical"):
        return Action.NOTIFY_HUMAN_THEN_RETRY
    return Action.AUTO_RETRY_WITH_LOG
Circuit Breaker — The HALT_AND_ESCALATE_TO_HUMAN path is the circuit breaker that prevents cascade failure.
Safe Default — AUTO_RETRY_WITH_LOG is the only scenario where blind auto-retry is acceptable.
